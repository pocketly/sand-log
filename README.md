## Sand Log Module
A Logger Module for use with the Sand Framework

## To begin

 1. Install it:

    ```bash
    $ npm install sand-log -S
    ```
 2. Require it and use:
 
     ```javascript
     var log = require('sand-log')('sand');
     log('It automatically applies the namespace');
     ```
